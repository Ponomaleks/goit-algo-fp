# Розрахунок ймовірностей сум при киданні двох кубиків

## Опис завдання
Метою цього експерименту було моделювання кидання двох ігрових кубиків та визначення ймовірностей отримання кожної можливої суми методом Монте-Карло. Для цього було проведено 100 000 симуляцій.

## Результати Монте-Карло
| Сума | Імовірність (МC) |
| ---- | ---------------- |
| 2    | 0.0274           |
| 3    | 0.0550           |
| 4    | 0.0826           |
| 5    | 0.1123           |
| 6    | 0.1397           |
| 7    | 0.1663           |
| 8    | 0.1375           |
| 9    | 0.1114           |
| 10   | 0.0835           |
| 11   | 0.0553           |
| 12   | 0.0290           |


## Аналітичні ймовірності
Відомо, що ймовірність отримання суми двох кубиків можна обчислити аналітично:

| Сума | Імовірність (аналітична) |
| ---- | ------------------------ |
| 2    | 1/36 ≈ 0.0278            |
| 3    | 2/36 ≈ 0.0556            |
| 4    | 3/36 ≈ 0.0833            |
| 5    | 4/36 ≈ 0.1111            |
| 6    | 5/36 ≈ 0.1389            |
| 7    | 6/36 ≈ 0.1667            |
| 8    | 5/36 ≈ 0.1389            |
| 9    | 4/36 ≈ 0.1111            |
| 10   | 3/36 ≈ 0.0833            |
| 11   | 2/36 ≈ 0.0556            |
| 12   | 1/36 ≈ 0.0278            |

## Висновки
1. Результати експерименту методом Монте-Карло **узгоджуються з аналітичними ймовірностями**.  
2. Найчастіше випадає сума **7**, що відповідає очікуваному розподілу ймовірностей.  
3. Чим більша кількість симуляцій, тим ближчі результати до аналітичних значень.  
4. Метод Монте-Карло підтверджує правильність розрахунків і може використовуватися для оцінки ймовірностей у складніших випадках, коли аналітичне рішення складно або неможливо отримати.
